Electromyography Decoding

Project Overview

This project analyzed the electromyography(EMG) signal of the amputee patient and compared the performance by implementing an algorithm that can handle the robotic arm precisely.

Timeline 2016
Role Implement two kinds of model
Tools MATLAB

Introduction

What is a Electromyography(EMG) Decoding?

  • The EMG signal is a biomedical signal that measures electrical currents generated in muscles

    • Algorithm implementation to precisely control the robotic arm by analyzing amputee patient's EMG signal

    Experimental Protocol
  • Based on the memory of hand motion before amputation, inference using EMG signals acquired while moving muscles related to hand motion
  • Collecting EMG signals from 8 channels of amputee patient (left)
  • Total 9 motions & 3 trials (right)

    • Method

    Feature extraction


  • Extracting 3 features; mean absolute, waveform length and variance in EMG signal
  • Left graphs indicate the features in Channel 1:

    • K-means clustering
  • Using to partition data into k clusters in a way that data points in the same cluster


  • Using the Mahalanobis distance which is a measure of the distance between a point and a distribution
  • Mahalanobis distance equation
    • Maximum Likelihood(ML)
  • Using to estimate the intended movement
  • ML Flow Chart (refer to. neural decoding page)
  • Using 24-dimensional (8 channels x 3 features) multivariate normal distribution
  • Assume the prior probabilities are equal

    Decoding performance
  • Confusion matrix results
  • Average decoding accuracies using

      • Maximum likelihood: 96.30%

    Conclusion

    Decoding performance
  • Propsed algorithm has a success rate of over 90%
  • So, it can be used for rehabilitation by applying it to the patient to control the robotic prosthesis in the future
    • Limitation
  • Since this result is a result of one subject, the result may change depending on the subject's motion proficiency
  • Therefore, verification with more diverse data is required
