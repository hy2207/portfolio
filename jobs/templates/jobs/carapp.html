    Design


  • Main rental page

  • User's car list page
      • Can check the list of cars saved by users
      • If user likes the car, he can swipe right to save it

  • Firebase
      • Load car data from firebase realtime database
  • Car stack view


      • Refers 'Tinder' application

      • User can swipe the image of car to right side if user likes the car, if not swipe to left side
      • The list what user swipe to right side will save to user list
